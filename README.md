# Schema Scribe

Schema Scribe is an open-source library designed to automatically generate human-readable documentation for your Sanity CMS database schemas. By configuring a simple setup and integrating it within your CI/CD pipeline, you can ensure your schema documentation remains up-to-date after every commit to your main branch.
